— TaxiHub · Release Notes

Versionshinweise.
Chronologisch.

Versionshinweise für die TaxiHub-Plattform und die Fahrer-App. Das Backend wird wöchentlich aktualisiert, App-Updates erscheinen alle drei bis vier Wochen.

v4.1.0

Update 2026-04-18

Live-ETA mit Verkehrsdaten und neuer Fahrer-Heatmap.

Neu

  • ETA-Berechnung berücksichtigt Echtzeit-Verkehrsdaten (Wien Straße/U-Bahn)
  • Heatmap für Fahrer: wo wird gerade am meisten bestellt
  • Fahrgast-App: Karten-Stil 'Nacht' für späte Fahrten

Korrigiert

  • Tip-Eingabe sprang gelegentlich auf 0 zurück

v4.0.0

Major Release 2026-02-01

Komplett neue Disponenten-Oberfläche und API v4.

Neu

  • Disponenten-Konsole mit Drag-and-Drop für Auftrags-Zuweisung
  • Webhooks für Buchhaltung (LexOffice, sevDesk, BMD)

Breaking Changes

  • API v3 läuft am 1. Juli 2026 aus — Migrationsleitfaden in den Docs

v3.4.0

Update 2025-11-20

Stripe Connect-Integration für direkte Fahrer-Auszahlungen.

Neu

  • Wöchentliche Auszahlungen direkt aufs Fahrer-Konto
  • Tagesabschluss-PDF automatisch per E-Mail

v3.0.0

Major Release 2025-06-04

Migration auf Kotlin Multiplatform für die Fahrer-App.

Neu

  • Geteilte Geschäftslogik zwischen iOS und Android
  • Offline-Modus: letzte 24 Stunden Aufträge bleiben verfügbar

v1.0.0

Major Release 2023-04-15

Launch mit 8 Wiener Taxi-Unternehmen.

Neu

  • Fahrer-App, Disponenten-Konsole, REST-API für Drittsysteme
